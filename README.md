## 四、六道轮回与因果

### 4.1 六道

**三善道：**
| 道 | 去向原因 |
|---|---------|
| 天道 | 积大功德或天人下凡 |
| 阿修罗道 | 有功德但嗔恨心重 |
| 人道 | 正常轮回，最适合修行 |

**三恶道：**
| 道 | 去向原因 |
|---|---------|
| 畜生道 | 愚痴/杀生多 |
| 饿鬼道 | 贪心重/债务多 |
| 地狱道 | 极恶之人 |

### 4.2 轮回驱动力：因果业力

**三世因果：**
- **命债**：伤人或杀人，需偿命
- **钱债**：借钱不还，需还钱
- **情债/桃花债**：感情纠葛，能量锁定极强

阿赖耶识（第八识）携带所有业力印记进入下一世，决定新生命的性格/天赋/命运。

### 4.3 冤亲债主

人欠债不还 → 债主变成冤亲债主，时刻跟随。

| 化解方式 | 效果 |
|---------|------|
| 直接还债 | 最根本 |
| 功德抵消 | 需足够大量 |
| 诚心道歉 | 取得债主原谅 |
| 诵经（辅助）| 积累功德，非决定性 |

### 4.4 超度与施食

- **超度**：必须债主主动放手才生效，不是诵经就能超度
- **施食**：给饿鬼能量，暂时缓解饥饿，不能超度
- **婴灵**：堕胎胎儿魂魄，能量极弱，需诚心道歉+功德回向

### 4.5 仙仍在轮回

仙人不是终点。福报享尽，重新轮回。只有穿过南天门才能彻底解脱。
